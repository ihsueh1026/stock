分點資料人工收件區 (broker branch manual input)
=================================================

放法：每檔一個檔案，命名 {股票代號}.txt
例：2330.txt、2454.txt …

內容：把你在券商/籌碼K線看到的「買超前N + 賣超前N 券商分點」直接貼進來即可。
格式不用整理，原始表格貼上就行（tab 或空白分隔都可以），我會寫 parser 解析。

我需要能辨識出來的最低欄位：
  - 券商分點名稱（例：凱基-台北、富邦-嘉義、外資券商名）
  - 該分點的「買賣超張數」（或同時有「買張」「賣張」也可，我會自己算淨額）

如果你的來源同時有「均價」也一併貼上，之後可估主力成本區。

第一行請標示日期（哪一天的分點），例：
  # 2026-05-19
若沒寫，預設用「昨天的交易日」。

==== 範例（格式示意，不必照抄）====
# 2026-05-19
買超
凱基-台北    1,200   100   +1,100   1085.0
摩根大通     980     0     +980     1086.5
...
賣超
美林         50      870   -820     1084.0
富邦-嘉義    20      640   -620     1083.5
...
